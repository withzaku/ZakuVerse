# Core Web Vitals Methology

## Core Objective

Define what to prioritize so page experience quality supports stronger search performance and better conversion outcomes.

## Web Research Signals Used

- Prioritize the current Core Web Vitals set: LCP, INP, and CLS.
- Prioritize passing performance thresholds at the 75th percentile.
- Prioritize field data as the main quality signal for user experience.
- Prioritize continuous monitoring because metrics and thresholds can evolve.

## Primary Ranking Priorities

- Prioritize loading performance quality for critical landing pages.
- Prioritize interactivity responsiveness on high-intent pages.
- Prioritize visual stability in above-the-fold and conversion sections.
- Prioritize mobile performance quality as a first-class target.
- Prioritize performance consistency across templates.
