# Image And Video Search Visibility Methology

## Core Objective

Define what to prioritize so image and video assets improve search discoverability, topical understanding, and conversion support.

## Web Research Signals Used

- Prioritize high-quality media near relevant contextual text.
- Prioritize descriptive image alt text tied to page meaning.
- Prioritize video pages with clear descriptive metadata signals.
- Prioritize media relevance to user intent and page purpose.
- Prioritize discoverability of media-first content surfaces.

## Primary Ranking Priorities

- Prioritize media assets on high-intent service and proof pages.
- Prioritize contextual alignment between media and supporting copy.
- Prioritize metadata quality for media-driven search appearance.
- Prioritize user confidence through clear visual relevance.
- Prioritize media assets that support decision-stage understanding.
