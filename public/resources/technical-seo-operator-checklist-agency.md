# Technical SEO Operator Checklist (Agency / Multi-Client)

Use this checklist for agencies managing multiple client properties.

## Core Focus

- Standardize crawl and index diagnostics across client portfolios.
- Prioritize high-impact route classes per client vertical.
- Keep remediation documentation consistent for reporting and handoff.

## Remediation Steps

- Build repeatable templates for canonical, redirect, and index control audits.
- Classify client URL groups into commercial, proof, utility, and low-value bins.
- Apply internal link reinforcement plans after each remediation sprint.

## Validation

- Compare crawl allocation improvements by client after each cycle.
- Track useful index ratio and key route stability across accounts.
- Report inquiry-quality and conversion-signal movement per engagement.
