# Technical SEO Operator Checklist (SaaS)

Use this checklist for SaaS websites with product, docs, blog, and pricing routes.

## Core Focus

- Prioritize crawl coverage for pricing, product, comparison, and integration pages.
- Protect docs architecture from duplicate and low-value route variants.
- Align trial and demo conversion routes with technical priority map.

## Remediation Steps

- Consolidate overlapping feature and use-case pages with weak differentiation.
- Stabilize canonicals across docs versions and changelog templates.
- Strengthen internal links from docs and blog to commercial intent pages.

## Validation

- Track crawl share of pricing and product routes weekly.
- Monitor useful index ratio on docs and solution pages.
- Compare qualified trial/demo inquiries pre and post remediation.
