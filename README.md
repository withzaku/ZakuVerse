## 🚀 Deployment

### Netlify

1. Connect your GitHub repo to Netlify
2. Set publish directory to `.` (root)
3. Leave build command empty
4. Deploy

### Vercel

1. Import the GitHub repo into Vercel
2. Framework: **Other**
3. Root directory: `.`
4. Deploy

## ⚙️ Customisation

Before going live, update these placeholders:

- `https://wa.me/your-number` → replace with your WhatsApp number (e.g. `https://wa.me/1234567890`)
- `hello@zakuverse.com` → your real email address
- Social media links in the footer
- Testimonial content with real client quotes
- Portfolio projects with your actual work
