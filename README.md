# ZakuVerse (React + Vite)

Production-ready multi-page startup website for **ZakuVerse**.

## Tech Stack

- React 19
- Vite
- React Router
- Custom CSS design system (futuristic dark UI)

## Pages

- `/` Home
- `/services` Services
- `/tools` Platform / Coming Soon Tools
- `/portfolio` Portfolio
- `/contact` Contact

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

- **Netlify**: uses `netlify.toml` (`npm run build`, publish `dist`)
- **Vercel**: uses `vercel.json` (`npm run build`, output `dist`)
