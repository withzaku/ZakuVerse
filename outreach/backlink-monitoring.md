# Backlink Monitoring Setup

## 1) Tool stack
- Primary: Google Search Console links report.
- Secondary: Ahrefs or Semrush (if available).
- Tertiary: manual monthly spot-check from referral analytics.

## 2) Baseline snapshot (Day 0)
- Total referring domains.
- Total backlinks.
- Follow vs nofollow ratio.
- Referring domains to /services/*, /resources/*, and /blog/*.
- Top anchor text clusters.

## 3) Weekly workflow
- Export new/lost backlinks.
- Tag each domain by relevance:
  - high relevance
  - medium relevance
  - low relevance
- Tag anchor quality:
  - branded
  - partial match
  - exact match
  - generic

## 4) Risk checks
- Sudden spikes of low-quality referring domains.
- Repetitive exact-match anchors.
- Sitewide links from unrelated domains.
- Links to non-canonical or redirecting URLs.

## 5) Attribution tracking
- Maintain a sheet with columns:
  - Date acquired
  - Source domain
  - Target URL
  - Anchor text
  - Outreach campaign name
  - Status (live/lost)

## 6) Monthly review metrics
- Net new referring domains.
- Net new links to commercial pages.
- Link growth by content type (service/resource/blog).
- Branded vs non-branded anchor trends.

## 7) Action triggers
- If lost high-relevance links > 5 in 30 days, start reclaim campaign.
- If low-relevance new links > 30% monthly share, tighten outreach list quality.
- If service-page link growth is flat for 60 days, prioritize case-study and resource promotion.
